# umami-api
REST API for cooks made with API Platform 

## Install

You need to have docker installed / everything needed for the install is in the docker-compose.

    $ git clone
    $ cd umami-api
    $ docker-compose up -d
    $ docker exec -it www bash

You can now go to https://localhost
